Installation auf Windows

  1. Lade die BlitzClient.exe herunter
  2. Führe die EXE-Datei aus (ggf. mit Rechtsklick → "Als Administrator ausführen")
  3. Folge den Installationsanweisungen
  4. Starte Minecraft und wähle den BlitzClient im Launcher aus
  5. Falls dein Antivirenprogramm warnt, füge eine Ausnahme hinzu

Tipp

Für die beste Performance empfehlen wir, Java 17 oder höher zu installieren.

Installation auf macOS

  1. Lade die BlitzClient.dmg herunter
  2. Öffne die DMG-Datei und ziehe BlitzClient in den Applications-Ordner
  3. Starte BlitzClient aus dem Applications-Ordner
  4. Falls eine Warnung erscheint: Rechtsklick → Öffnen (nur beim ersten Mal nötig)
  5. Wähle den BlitzClient im Minecraft Launcher aus

Tipp

Für Apple Silicon (M1/M2) wird der Client automatisch optimiert.

Installation auf Linux

  1. Lade die BlitzClient.AppImage herunter
  2. Mache die Datei ausführbar: chmod +x BlitzClient.AppImage
  3. Führe die AppImage aus: ./BlitzClient.AppImage
  4. Falls nötig, installiere fehlende Abhängigkeiten (libfuse2 etc.)
  5. Wähle den BlitzClient im Minecraft Launcher aus

Tipp

Für bessere Integration kannst du die AppImage mit --install installieren.
